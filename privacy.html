Privacy Policy

Effective date: April 3, 2026

1. Data We Collect

Kalshi Signal collects TikTok OAuth tokens (an access token and a refresh token) for the sole purpose of posting videos to the authorized TikTok account. The only personal identifier received is the TikTok user open_id provided during the OAuth authorization flow.

We do not collect any additional personal data. We do not use tracking technologies, analytics cookies, or advertising identifiers.

2. How We Use Your Data

OAuth tokens are used exclusively to publish prediction market analysis videos to the authorized TikTok account via the TikTok Content Posting API. No data is used for any other purpose.

3. How We Store Your Data

All data is stored locally on the app operator's machine in an encrypted SQLite database. Data is not stored in the cloud or on any remote server. Refresh tokens are encrypted at rest. Access tokens are automatically refreshed, and expired tokens are overwritten.

4. Data Retention

OAuth tokens are retained only as long as needed to maintain the posting integration. Users can revoke access at any time through their TikTok account settings (Settings → Security → Manage app permissions), which immediately invalidates all stored tokens.

5. Third-Party Services

Kalshi Signal uses the Kalshi public API to retrieve prediction market data. No user data is shared with Kalshi. The app also uses Anthropic's Claude API for content research and script generation. No TikTok user data is sent to Anthropic.

6. Contact

For privacy inquiries, contact us at mineralboy1999@gmail.com.

7. Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be posted on this page with an updated effective date.
